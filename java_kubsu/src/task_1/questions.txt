1. Объясните основное назначение метода main().
Это точка входа в программу на Java. Без него программа не запустится, так как именно с него начинается выполнение кода.

2. Расскажите о входном параметре метода main() String[] args.
Это массив строк, который используется для передачи аргументов в программу при её запуске из командной строки.

3. В чем состоит разница между JDK (Java Development Kit) и JRE (Java Runtime Environment)?

JDK – это набор инструментов для разработки (компилятор, библиотеки, утилиты).
JRE – это только среда для запуска программ (включает JVM и библиотеки).

4. Какое свойство массива возвращает его размер?
Для получения размера массива используется свойство array.length.

5. В чем отличие конструкций System.out.println() и System.out.print()?

println() выводит текст и переносит курсор на новую строку.
print() просто выводит текст без перехода на новую строку.

6. Каким образом осуществляется конкатенация строк в Java?
Используется оператор +, например: "Hello" + " World".

7. Что означает ключевое слово void?
Обозначает, что метод не возвращает никакого значения.

8. Что означает ключевое слово static?
Означает, что метод или переменная относятся к классу, а не к объекту. Можно вызывать метод без создания объекта.
